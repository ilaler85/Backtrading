29. Задача о пробирках. Даны три пробирки объемом 100 мл. Две из них имеют
шкалу рисок. Уровни рисок указываются пользователем. Из начального
состояния {100 мл, 0 мл, 0 мл} (в первой пробирке 100 мл, а две другие
пусты) определить, возможен ли переход путем переливания жидкости из
одной пробирки в другую в состояние { _ , _ , 1 мл} (в пробирках с
рисками — произвольное количество миллилитров, а в пробирке без
рисок — 1 мл).
